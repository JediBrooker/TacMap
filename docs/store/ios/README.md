# App Store screenshots

Captured from the real app on the iOS Simulator at **Shoalwater Bay Training
Area, QLD**, via the `TacticalMapsUITests` screenshot UI test (see
`scripts/ios_screenshots.sh` notes below). Regenerate with that test; extract
attachments from the `.xcresult` with `xcrun xcresulttool export attachments`.

## iPhone — `iphone-6.9/` (1320×2868, iPhone 6.9")
Mandatory size for App Store Connect. Upload 3–10.
1. `01-main.png` — live MGRS HUD over the coastline
2. `02-symbol-builder.png` — NATO APP-6C symbol builder (affiliation / echelon / function + live preview)
3. `03-symbols-on-map.png` — APP-6 symbols + an Assembly Area task on the map
4. `04-drawings.png` — drawing tools (line / area / point)
5. `05-about.png` — data sources, attributions, on-device-data note

## iPad — `ipad-13/` (2064×2752, iPad 13")
Required because the app is universal (iPhone + iPad). Same five screens.

## Notes
- Generated by the `TacticalMapsUITests` UI-test target (added to
  `ios/project.yml`). Location is set at the device level by
  `xcrun simctl location set` so the basemap shows Shoalwater Bay.
- Known polish items (optional): `03`/`04` have a symbol's controls card open at
  the bottom; `03` shows two friendly units + a task rather than friend-vs-
  hostile (the Hostile affiliation tap didn't register in the run).
