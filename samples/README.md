# samples/

This directory is intentionally empty in the public repo.

The Holsworthy North 1:25,000 PDF that was used during development is
Australian Defence Force / AUSLIG copyright and **must not be redistributed**.
It was removed before the App Store submission per
`docs/APPSTORE_CHECKLIST.md`.

If you need a freely-redistributable GeoPDF for testing, the USGS publishes
the entire US Topo series under public domain:

  https://www.usgs.gov/programs/national-geospatial-program/us-topo-maps-america

Drop any `.pdf` you download into this directory; `project.yml` resources the
`samples/` folder so it ships into the app bundle for testing.
